# 🧠 Math Quiz Game (C++)

## 📖 Overview
This project is a console-based **Math Quiz Game** developed in C++ that challenges users with randomly generated arithmetic questions.

The player can choose difficulty levels and operation types, and the system tracks performance across multiple questions.

---

## 🚀 Features
- 🔢 Random math question generation
- 🎯 Multiple difficulty levels:
  - Easy
  - Medium
  - Hard
  - Mixed
- ➕ Multiple operation types:
  - Addition
  - Subtraction
  - Multiplication
  - Division
  - Mixed
- 📊 Score tracking (correct vs wrong answers)
- 🎨 Visual feedback using console colors
- 🔁 Replay option

## ⚙️ How It Works

1. User selects:
   - Number of questions
   - Difficulty level
   - Operation type

2. For each question:
   - Two random numbers are generated
   - Operation is selected
   - User inputs answer
   - System checks correctness

3. At the end:
   - Total questions displayed
   - Correct and wrong answers counted
   - Game summary shown

4. Player can choose to play again.
